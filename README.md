# How to use

1. Select the device for which you need a picture, and find its model
   
<img width="1406" alt="Знімок екрана 2023-10-05 о 19 23 43" src="https://github.com/xrust83/iconz2m/assets/58448955/e0be6016-7568-4a95-891b-52314cf8f39a">

2. Check for the image in the images folder of this repository
   
<img width="1195" alt="Знімок екрана 2023-10-05 о 19 27 07" src="https://github.com/xrust83/iconz2m/assets/58448955/8f15f4ed-88d9-4e70-86be-e878f99c682d">

3. Go to the device settings

<img width="1062" alt="Знімок екрана 2023-10-05 о 19 29 08" src="https://github.com/xrust83/iconz2m/assets/58448955/8065a97e-c84c-4975-97fd-39955f3eeef4">

4.At the bottom of the page, in the "icon" field, enter a link to the picture `https://raw.githubusercontent.com/xrust83/iconz2m/main/images/NAME.png` where NAME is the model of your device

<img width="1076" alt="Знімок екрана 2023-10-05 о 19 30 23" src="https://github.com/xrust83/iconz2m/assets/58448955/92657a8c-1217-41a4-b3ac-678aef3fca52">

5. But save by pressing submit
